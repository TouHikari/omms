**[旁白]**
老师、同学们，大家好！我们是 **[你的团队名称]**。今天非常荣幸能为大家展示我们的《软件工程》实验大作业——**OMMS 在线医疗管理系统**。

OMMS 是一个基于 **前后端分离** 架构的全功能医疗管理平台。它旨在为医院提供从患者预约、医生接诊、电子病历管理到药房库存控制的一站式解决方案。

**[旁白]**
在技术选型上，我们追求高效与稳定。
后端采用 Python 的 **FastAPI** 框架，配合 **SQLAlchemy** 和 **MySQL** 数据库，确保了高性能的数据处理能力。
前端使用 **Vue 3** 全家桶，搭配 **Ant Design Vue** 组件库和 **Vite** 构建工具，打造了流畅的现代化用户体验。
此外，我们实现了系统的**完全上云**，前后端及数据库均部署在**华为云**远程服务器上，并使用**宝塔面板**进行可视化运维管理。

## 2. 核心功能演示 (1:00 - 5:30)

### 场景一：患者预约挂号

**[画面]**
- 打开浏览器，访问登录页 `/login`。
- 点击“注册账户”，快速演示注册流程（输入用户名、密码、邮箱）。
- 或者直接使用已有的患者账号登录。

**[旁白]**
首先，让我们从**患者**的视角开始。
患者可以通过门户网站注册账户。登录后，系统会自动跳转到患者专属的仪表盘。

**[画面]**
- 登录成功，进入 `/appointments` 页面。
- 点击侧边栏的“新建预约” (如果侧边栏有的话，或者页面上的按钮)。
- 在预约表单中：选择“内科”，选择医生（例如“李医生”），选择日期和时间段。
- 点击“提交预约”。

**[旁白]**
患者最核心的需求是**预约挂号**。在“预约管理”模块，患者可以直观地查看到各科室医生的排班情况。
这里我们演示一下：选择“内科”，指定“李医生”，预约明天上午的时间段。系统会实时检查号源冲突，确认无误后，预约即刻生效。

**[画面]**
- 页面跳转回预约列表，显示刚才的新增记录，状态为“待就诊”。

**[旁白]**
提交成功后，预约状态显示为“待就诊”，患者可以随时查看详情或取消预约。

### 场景二：医生接诊与病历录入

**[画面]**
- 退出当前账号。
- 使用医生账号登录（例如 `doctor/123456`）。
- 进入 `/` (Dashboard) 或直接进入 `/appointments`。

**[旁白]**
接下来，我们切换到**医生**视角。医生登录后，可以在工作台看到今日的待诊患者列表。

**[画面]**
- 找到刚才患者的那条预约记录。
- 点击“接诊”或“查看详情”按钮（假设有此功能，或者模拟流程）。
- 切换到 `/records` (病历管理) -> “新建病历”。
- 在新建病历页面：选择刚才的患者，填写主诉（如“头痛、发热”），诊断（如“上呼吸道感染”）。
- **重点**：在“处方信息”区域，点击“添加药品”，选择“阿莫西林”，输入数量“2盒”。

**[旁白]**
医生在接诊时，可以快速调阅患者的历史病历。
在本次就诊中，医生录入患者的主诉“头痛发热”，诊断为“感冒”。
最关键的是**处方开具**环节。系统集成了药品库存数据，医生在开药时，系统会自动校验库存，避免开出缺货药品。这里我们开具两盒“阿莫西林”。

**[画面]**
- 点击“保存病历”或“提交”。
- 提示“保存成功”。

**[旁白]**
病历提交后，电子处方会自动流转到药房系统。

### 场景三：药房发药与库存管理

**[画面]**
- 退出医生账号。
- 使用药房管理员/护士账号登录。
- 进入 `/pharmacy` (药房管理)。
- 点击“处方列表” -> “待审核/待发药”。

**[旁白]**
现在是**药房管理员**的操作界面。
在“处方管理”中，我们可以看到医生刚刚开具的处方。药师审核无误后，点击“发药”。

**[画面]**
- 找到刚才的处方，点击“发药”或“确认”。
- 切换到“库存列表” (Inventory)。
- 搜索“阿莫西林”，展示当前的库存数量。

**[旁白]**
发药操作会触发后台的**库存扣减**逻辑，保证账实相符。
同时，系统具备**智能预警**功能。当药品库存低于设定的安全阈值，或药品临近有效期时，系统会在仪表盘醒目位置发出警报，提示及时补货。

### 场景四：管理员报表统计

**[画面]**
- 切换为管理员账号 (Admin)。
- 进入 `/reports` (报表管理) 或 `/` (Dashboard)。
- 展示 ECharts 图表：就诊人数趋势图、药品销量占比饼图等。

**[旁白]**
最后是**管理员**视角。
OMMS 提供了强大的数据可视化能力。通过集成 **ECharts**，我们可以实时查看每日的就诊流量趋势、热门药品排行以及科室收入占比。
这些数据报表支持一键导出 CSV，为医院的管理决策提供有力支持。
